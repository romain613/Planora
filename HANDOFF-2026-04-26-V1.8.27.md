## 1. Contexte du bug

Console browser remontait régulièrement :
```
[CONTACT SAVE] OK ...
PUT https://calendar360.fr/api/voip/settings 403 Forbidden
```

Le 403 n'était pas causé par le save contact (temporalité console trompeuse). Cause réelle :
- Backend route `PUT /api/voip/settings` exige `requireAdmin + enforceCompany`
- Frontend `togglePhoneRecording` (CollabPortal:413) appelait ce PUT
- Toggle visible pour TOUS les collaborateurs dans PhoneTab (lignes 5917 + 7002)
- Quand un collab non-admin (`role='member'`) toggle l'enregistrement → backend rejette → 403 silencieux logué dans console

**Le setting `voip_settings.recordingEnabled` est company-wide** (clé `companyId` en DB). Logique normale qu'il soit réservé aux admins. Le bug était UI.

## 2. Fix appliqué (3 patches frontend, 0 backend, 0 DB)

### 2.1 CollabPortal.jsx — handler avec garde admin
[`app/src/features/collab/CollabPortal.jsx:413-427`](app/src/features/collab/CollabPortal.jsx#L413)

```diff
   const togglePhoneRecording = () => {
+    // V1.8.27 — recordingEnabled est un setting company-wide (voip_settings keyed by companyId).
+    // Backend route PUT /api/voip/settings exige requireAdmin → fix 403 silencieux pour members.
+    const _isAdmin = collab?.role === 'admin' || collab?.role === 'supra' || isAdminView;
+    if (!_isAdmin) {
+      showNotif("Réservé à l'administrateur", "danger");
+      return;
+    }
     const next = !phoneRecordingEnabled;
     setPhoneRecordingEnabled(next);
     localStorage.setItem("c360-phone-record-"+collab.id, next?"1":"0");
     api('/api/voip/settings', { method:'PUT', body:{ recordingEnabled: next, recordingConsent: next }}).catch(()=>{});
     showNotif(next ? "Enregistrement d'appels active 🎙️" : "Enregistrement desactive");
   };
```

### 2.2 PhoneTab.jsx — flag `_isAdminPhone` + 2 toggles grisés
[`app/src/features/collab/tabs/PhoneTab.jsx:138`](app/src/features/collab/tabs/PhoneTab.jsx#L138) (flag) + lignes 5920 (toggle 1) + 7005 (toggle 2)

- Flag `_isAdminPhone = collab?.role === 'admin' || collab?.role === 'supra' || isAdminView` calculé en tête du composant
- Toggle haut "Enregistrement auto ON/OFF" (ligne 5920) : `cursor:not-allowed` + `opacity:0.5` + tooltip "Réservé à l'administrateur" + `onClick={undefined}` si non-admin
- Toggle Card "Enregistrement auto" (ligne 7005) : idem + `onClick=showNotif` au lieu du toggle local pour les members

### 2.3 Defense in depth (2 niveaux)
1. **UI** : toggles grisés visuellement pour members
2. **Handler** : garde admin dans `togglePhoneRecording` même si appelé par un autre call site
